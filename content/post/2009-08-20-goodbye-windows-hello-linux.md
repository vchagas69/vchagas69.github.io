After several (failed) attempts of adopting Linux as my main Operating System, I finally found a friendly enough Linux distro for a long time Microsoft user.

![Puppy Linux 4.20](/post/2009-08-20-goodbye-windows-hello-linux_files/9334687_6BjYm.png)

It is Puppy Linux (http://www.puppylinux.org) and is currently on release 4.20.

Download the CD image from the URL below.

http://distro.ibiblio.org/pub/linux/distributions/puppylinux/puppy-4.2-k2.6.25.16-seamonkey.iso

My adventures with this OS will definitely produce interesting writing material.
